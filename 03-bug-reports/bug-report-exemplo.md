# Bug Report

**Título:** Sistema permite login sem senha  
**Ambiente:** Web – Chrome  
**Severidade:** Alta  
**Prioridade:** Alta  

## Passos para Reprodução
1. Acessar tela de login
2. Informar e-mail válido
3. Deixar o campo senha vazio
4. Clicar em "Entrar"

## Resultado Esperado
Sistema deve bloquear o acesso e exibir mensagem de erro.

## Resultado Obtido
Sistema permite login sem informar senha.
