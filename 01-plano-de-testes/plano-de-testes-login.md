# Plano de Testes – Funcionalidade de Login

## 1. Objetivo
Validar o correto funcionamento da funcionalidade de login da aplicação web, garantindo que usuários consigam acessar o sistema apenas com credenciais válidas.

## 2. Escopo
- Login com dados válidos
- Login com dados inválidos
- Validação de campos obrigatórios

## 3. Fora do Escopo
- Testes de performance
- Testes de segurança avançados

## 4. Ambiente de Testes
- Aplicação Web
- Navegador: Google Chrome
- Sistema Operacional: macOS

## 5. Critérios de Entrada
- Funcionalidade desenvolvida
- Ambiente disponível para testes

## 6. Critérios de Saída
- Todos os casos de teste executados
- Bugs críticos reportados

## 7. Riscos
- Requisitos incompletos
- Falta de ambiente estável
